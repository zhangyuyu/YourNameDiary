# Your Name Diary

## 需求
你的名字，同款日记Web Application

## 技术栈
* nodejs
* mongodb
* bootstrap
* gulp

## 功能
* Add a diary
* Record time to diary
* Add a photo to diary
* Add location to diary
* Add weather to diary
* Add emotion to diary
* Delete diary
* View diary list
* Emoji

### Build the Application
npm install

### Start the Application
gulp start

### Access the Application
http://localhost:3000/
